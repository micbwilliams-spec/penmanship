How this works

The Protocol

One device per week. Twenty minutes a day, inside the journaling slot you already have. Don't stack weeks, don't skip ahead.

The weekly loop

  1. Read the definition once. Don't study it — just understand it.
  2. Listen to the example track and catch the device in the wild. Then spot it in songs you already love all week.
  3. Write four bars using the device. Save them in the app. Wrong attempts still build the muscle.
  4. Mark complete only when the bars are written. Listening doesn't count as done.

Rules of engagement

Re-read your saved bars at the end of every phase (weeks 5, 10, 15, 20). The cringe is the measurement — it means the writer has outgrown the writing.

If a device feels easy, you're probably doing it on autopilot. Push for one version that surprises you.

By week 16 you're making album-level decisions for There's Hope in the Music, not just bar-level ones. Slow down there.

Priority rule: resits outrank this. If a week falls during August exams, pause the schedule. Never stack two devices to catch up.
